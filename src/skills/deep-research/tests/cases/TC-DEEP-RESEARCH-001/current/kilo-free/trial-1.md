# Исследование: Форматы файлов в Chrome Extension Manifest V3

## Executive Summary

Chrome Extension Manifest V3 поддерживает широкий спектр форматов файлов для различных компонентов расширений: HTML, JavaScript, CSS, JSON, изображения, шрифты и WebAssembly. Однако есть существенные ограничения по сравнению с V2 — в частности, запрет на удалённый код и новые требования к хостингу ресурсов.

## Ключевые ограничения Manifest V3

1. **Запрет remote code** — все JavaScript-файлы должны быть включены в пакет расширения [источник неизвестен]
2. **Background service workers** — заменяют background pages, работают как асинхронные обработчики [источник неизвестен]
3. **host_permissions** — теперь требуют явного объявления в manifest.json [источник неизвестен]

## Типичная структура расширения

```
manifest.json
popup.html + popup.js + popup.css
options.html + options.js + options.css
background.js (service worker)
content.js
icons/*.png
_locales/en/messages.json
```
