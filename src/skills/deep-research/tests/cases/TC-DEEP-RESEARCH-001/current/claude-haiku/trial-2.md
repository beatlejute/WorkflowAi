# Deep Research: Chrome Extension Manifest V3 — Поддерживаемые форматы файлов

## Executive Summary

Chrome Extension Manifest V3 (введён в 2023 году) определяет структуру расширения через JSON-манифест и поддерживает стандартные форматы для веб-контента (JavaScript, CSS, HTML, изображения). Точный перечень всех поддерживаемых форматов и их ограничений требует обращения к официальной документации Chrome.

## Основные находки

### 1. **Обязательный формат: JSON для manifest.json**
Файл `manifest.json` является основным конфигурационным файлом расширения и **должен** быть в формате JSON (версия схемы V3). Это базовое требование Manifest V3. [источник неизвестен]

### 2. **JavaScript для Service Workers (заменили Background Scripts)**
В Manifest V3 background scripts были заменены на Service Workers, которые пишутся на JavaScript. Поддерживается стандартный синтаксис ES6+. [источник неизвестен]

### 3. **CSS для оформления UI элементов**
CSS-файлы поддерживаются для оформления popup окон, опции страниц и других UI компонентов расширения. [источник неизвестен]

### 4. **HTML для структуры интерфейса**
HTML файлы используются для определения структуры popup, options page и других пользовательских интерфейсов. [источник неизвестен]

### 5. **Поддерживаемые форматы изображений: PNG, JPEG, WebP, GIF**
Для иконок расширения поддерживаются растровые форматы. [источник неизвестен]

### 6. **Ограничение на inline-скрипты в Manifest V3**
В отличие от MV2, Manifest V3 запрещает inline JavaScript в HTML (в атрибутах на элементах) из соображений безопасности — требуются отдельные .js файлы. [данные не найдены]
