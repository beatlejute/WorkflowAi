# Шаблон: Result

Заполняется исполнителем после выполнения задачи. Используй при создании секции `## Result` в тикете.

```markdown
## Result

### Что сделано
- {описание выполненной работы по каждому пункту DoD}

### Изменённые файлы
- {путь/к/файлу} — {что изменено}
- {путь/к/файлу} — {что создано}

### Заметки
- {контекст для связанных задач, обнаруженные проблемы, обоснования решений}
```

## Правила заполнения

| Поле | Обязательно | Что писать |
|------|-------------|------------|
| `### Что сделано` | **Да** | Конкретные действия, evidence по каждому пункту DoD. Не шаблонный текст |
| `### Изменённые файлы` | **Да** (если DoD требует изменений) | Каждый созданный/изменённый файл с описанием правки |
| `### Заметки` | Нет | Дефекты, блокеры, контекст для следующих задач |
