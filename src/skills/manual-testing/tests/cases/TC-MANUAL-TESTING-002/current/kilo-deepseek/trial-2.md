## TC-001: «Кнопка отображается корректно с иконкой и текстом»

**Статус:** BLOCKED

**Причина:** Инструмент тестирования (браузер/Playwright MCP) недоступен в данной среде для выполнения визуальной проверки. Для подтверждения визуального критерия необходим скриншот элемента, который не может быть получен.

**Evidence предоставлено:** DOM-элемент найден, querySelector вернул элемент. Однако это недостаточно для проверки визуального отображения (иконка и текст).

**Шаги для ручного воспроизведения:**
1. Откройте приложение в браузере.
2. Найдите кнопку по селектору.
3. Убедитесь, что кнопка отображается с иконкой и текстом.
4. Сделайте скриншот.

**Рекомендация:** Для выполнения визуального тестирования необходим доступ к инструменту браузера (Playwright MCP) или возможность делать скриншоты.
