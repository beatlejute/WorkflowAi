### Шаги для ручного воспроизведения
1. Откройте тестируемую страницу в браузере.
2. Найдите кнопку по селектору (указать конкретный селектор из предоставленного evidence).
3. Визуально убедитесь, что:
   - Кнопка отображается (не скрыта, не обрезана).
   - На кнопке присутствует текст (соответствует ожидаемому).
   - Рядом с текстом (или внутри кнопки) отображается иконка.
   - Стили кнопки соответствуют макету (цвет, размер, выравнивание).
4. Сделайте скриншот области с кнопкой и сохраните как `reports/qa001-screenshot-01.png`.
5. Откройте скриншот и проверьте, что видимое состояние соответствует критерию «кнопка отображается корректно с иконкой и текстом».

### Рекомендация
Для выполнения полной проверки TC‑001 требуется доступ к инструменту тестирования UI (браузер с возможностью скриншотов). Если критичность сценария высока — создайте `HUMAN`‑тикет для ручного тестирования.
