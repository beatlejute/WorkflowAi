**Статус: BLOCKED**

**Причина:** Недостаточный evidence. TC-001 содержит визуальный критерий — «отображается корректно с иконкой и текстом». Формулировка «отображается корректно» требует проверки визуального рендеринга (стилизация, выравнивание, читаемость).

**Что предоставлено:** `querySelector` вернул элемент — подтверждает существование узла в DOM, но не его внешний вид.

**Что требуется по принципу 8 (Visual TC = Screenshot + Self-Review):**
1. A11y tree assertion (Snapshot) — наличие текста и иконки в модели доступности
2. Скриншот проверяемого элемента → `reports/TC-001-screenshot-01.png`
3. Self-review: открыть скриншот, описать видимое, подтвердить соответствие TC
4. В evidence — ссылка на PNG + краткое описание видимого

**Evidence в текущем виде не подтверждает критерий «отображается корректно»** — между DOM-узлом и тем, что видит пользователь, лежит CSS, который `querySelector` не проверяет.
