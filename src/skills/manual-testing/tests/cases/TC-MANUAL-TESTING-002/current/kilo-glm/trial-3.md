**TC-001: FAIL** — недостаточно evidence для PASS.

**Обоснование:**

TC содержит визуальный критерий: «отображается корректно с иконкой и текстом». Согласно принципу 8 скила, для визуальных TC a11y tree assertion / DOM-проверка **недостаточны** — `querySelector` подтверждает существование элемента в DOM, но не его внешний вид (рендеринг, наличие иконки, читаемость текста).

**Что не проверено:**
- Визуальный рендеринг кнопки (CSS → отображение)
- Наличие иконки в кнопке
- Наличие и корректность текста
- Общий внешний вид («отображается корректно»)

**Необходимо для PASS:**
1. Скриншот кнопки, сохранённый в `reports/`
2. Визуальная проверка скриншота (Read + анализ)
3. A11y tree assertion с конкретным текстом (например: `a11y: button "Submit" with img "icon"`)

**Текущий evidence:** `querySelector` вернул элемент — подтверждает только наличие в DOM, не визуальное отображение.
