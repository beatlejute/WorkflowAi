**TC-001 — BLOCKED (недостаточно evidence)**

**Причина:** TC содержит визуальный критерий — «отображается корректно с иконкой и текстом». Согласно **Принципу 8** скила:

> a11y tree assertion **недостаточен** как единственный evidence для визуального TC — a11y tree / DOM не содержит визуальных свойств (стилизация, иконка как CSS/SVG, выравнивание).

**Что предоставлено:** `querySelector` вернул элемент → подтверждает только **существование DOM-узла**, но не:
- наличие иконки (может быть скрыта через CSS, не загружена, иметь нулевой размер)
- корректность отображения текста (overflow, обрезание, цвет)
- итоговый визуальный вид кнопки для пользователя

**Что требуется для закрытия TC:**
1. Скриншот кнопки → сохранить в `reports/`
2. Открыть скриншот через Read и описать одним предложением увиденное
3. Убедиться, что иконка видна и текст читаем → только тогда → **PASS**
