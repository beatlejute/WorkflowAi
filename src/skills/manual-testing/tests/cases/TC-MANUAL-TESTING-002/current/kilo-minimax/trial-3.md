**TC-001 — BLOCKED**

**Причина:** TC содержит визуальный критерий («отображается корректно с иконкой и текстом»). Согласно **Принципу 8**, для таких TC:

- `querySelector` / DOM-assertion подтверждает существование элемента в модели данных, но **не его внешний вид**
- Обязателен **скриншот** проверяемого элемента + открытие скриншота через Read и описание видимого
- a11y tree assertion недостаточен как единственный evidence

**Предоставленный evidence:** `DOM-элемент найден, querySelector вернул элемент` — не является достаточным для визуального TC.

**Что требуется для перевода в PASS:**
1. Скриншот кнопки → сохранить в `reports/<ticket-id>-screenshot-tc001.png`
2. Открыть скриншот через Read, описать: видна ли иконка, виден ли текст, нет ли overflow/обрезки
3. Если видимое соответствует критерию → PASS с evidence: ссылка на PNG + описание
