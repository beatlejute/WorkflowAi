Директория `plans/current/` пуста — план для декомпозиции не найден.

---

## Итог: Инфраструктурная ошибка — отсутствует план

Обнаружено два критических условия, при которых декомпозиция **не может быть продолжена**:

1. **`plan_path` не передан** в инструкциях → неизвестно, какой план декомпозировать.
2. **`plans/current/` пуст** → нет доступного плана.

### Что сделано корректно

- ID получены через `node .workflow/src/scripts/get-next-id.js --all-from-config` — машинная аллокация, не ручной подсчёт.
- Карта диапазонов: `{"ARCH":1,"IMPL":1,"FIX":1,"REVIEW":1,"DOCS":1,"ADMIN":1,"QA":1,"COACH":1,"HUMAN":1,"RSH":1}`
- Ручное вычисление «взять максимальный ID из archive + 1» **не применялось**. Даже при наличии информации об архивных тикетах IMPL-001..005 — скрипт `get-next-id.js` является единственным допустимым источником нумерации.

### Следующий шаг

Передать `plan_path` в Instructions, например:

```
plan_path: .workflow/plans/current/PLAN-001.md
```

После этого декомпозиция может быть запущена повторно.
